# Parallax AI Logo & Branding Assets

## Logo Files

### `parallax-logo-original.png`
- **Usage**: Full logo for splash screens, about pages, marketing materials
- **Style**: Watercolor-style illustration with interlocking geometric blocks in a speech bubble
- **Colors**: Teal/blue outer ring, tan/gold and teal interlocking blocks, directional arrows
- **Dimensions**: Original watercolor artwork

### `parallax-icon.svg`
- **Usage**: Standard icon for app UI, toolbars, medium-sized displays
- **Dimensions**: 64x64px
- **Style**: Simplified vector version maintaining the core geometric pattern
- **Features**:
  - Hexagonal outer frame
  - Interlocking 3D blocks (teal and tan)
  - Directional arrows indicating perspective/parallax effect

### `parallax-icon-simple.svg`
- **Usage**: Favicon, small UI elements, mobile app icon
- **Dimensions**: 32x32px
- **Style**: Ultra-simplified version for maximum clarity at small sizes
- **Features**:
  - Minimal hexagon frame
  - Simplified block pattern
  - Single directional arrow

## Usage Guidelines

1. **Primary Logo**: Use `parallax-logo-original.png` for:
   - Landing pages
   - About/Help sections
   - Marketing materials
   - Large format displays

2. **Standard Icon**: Use `parallax-icon.svg` for:
   - App header/navigation
   - Desktop application icon
   - Social media profiles (large)

3. **Favicon**: Use `parallax-icon-simple.svg` for:
   - Browser tab icon (convert to .ico)
   - Mobile app icon
   - Small UI indicators
   - Social media profiles (small)

## Geometric Symbolism

The interlocking blocks represent:
- **Multiple perspectives** (the core "parallax" concept)
- **Technical precision** (geometric forms)
- **Interconnected thinking** (interlocking pieces)
- **Dimensional analysis** (3D representation)

The hexagonal frame and speech bubble suggest communication and structured thinking.
